# E1RM Calculator

RPE-based one rep max calculator, available on Android and iOS.

## Repository Structure

```
e1rm-calculator/
├── android/   # Android app (Kotlin, Jetpack Compose)
└── ios/       # iOS app (Swift, SwiftUI)
```

## Android

See [`android/`](android/) for build instructions.

**Play Store**: [E1RM Calculator](https://play.google.com/store/apps/details?id=com.e1rm.calculator)

## iOS

See [`ios/`](ios/) for build instructions.

**App Store**: [E1RM Calculator — RPE Based](https://apps.apple.com/us/app/e1rm-calculator-rpe-based/id6759536865)
